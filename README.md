# YouTube Video Trend Exploratory Data Analysis (EDA)

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a YouTube trending videos dataset.  
The dataset contains information about video views, likes, dislikes, comments, channels, and publishing trends across USA, Canada, and Great Britain.

The goal is to analyze user engagement patterns and video performance trends.

---

## Objectives
- Understand dataset structure and missing values
- Analyze video performance metrics such as views, likes, dislikes, and comments
- Study publishing trends over time
- Identify top and bottom performing channels
- Explore relationship between views and dislikes
- Extract meaningful insights using visualization techniques

---

## Dataset Information
The dataset includes the following features:

- Video ID  
- Title  
- Channel ID  
- Channel Title  
- Published At  
- View Count  
- Likes  
- Dislikes  
- Comment Count  
- Tags  
- Description  
- Comments  

---

## Tools and Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## Key Analysis Performed
- Data cleaning and handling missing values
- Date-time conversion and feature engineering
- Extraction of publishing month
- Monthly video upload analysis
- Top 10 and bottom 10 channels
- Most liked and most disliked videos
- Relationship analysis between views and dislikes

---

## Key Insights
- Certain channels dominate video uploads and engagement
- Views and dislikes show a weak to moderate relationship
- Upload activity varies across different months
- A small number of videos receive significantly higher engagement
